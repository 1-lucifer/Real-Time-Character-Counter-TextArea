# Real-Time Character Counter Text Area

A simple and interactive text area project built using HTML, CSS, and JavaScript that displays a real-time character counter while the user types.

## Features

- Real-time character counting
- Displays typed characters and remaining characters
- Prevents typing beyond the maximum limit
- Warning message when limit is reached
- Responsive and modern UI design
- Smooth styling and shadow effects

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Functionality

- Maximum character limit: `200`
- Updates counter instantly while typing
- Stops additional input after reaching the limit
- Displays warning message when limit is reached
